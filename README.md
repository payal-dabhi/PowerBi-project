## Project objective
Owner of Madhav Store wants us to help them create a dashboard to track and analyse their online sales across india.
## Project key point
-	Data collection
-	Data cleaning & wrangling
-	Data modelling
-	Data visualization
-	Dashboard design
-	Sharing Insights
